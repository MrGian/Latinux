<h1 align="center">Latinux</h1>
<h3 align="center">
L'app per il latino
</h3>

Latinux **era** un'app che permetteva la consultazione di esercizi svolti di latino e di un dizionario latino-italiano.

L'app **non** è più disponibile per i nuovi utenti.
